SERVICES

User Authentication:
  • Sign-Up and Login: Users can easily sign up and log in to the application using a secure authentication system.
  • OAuth Integration: The application supports login via Google and GitHub, making it convenient for users who prefer these platforms.
    • Contact Management:
  • Save Contacts: Users can add new contacts to their list, including all relevant information such as names, phone numbers, emails, and addresses.
  • View Contacts: A clean and organized interface allows users to view their saved contacts at a glance.
  • Favorite Contacts: Users can mark important contacts as favorites for quick access.
  • Detailed Contact Information: The application enables users to save detailed information about each contact, including images, official data, and personal notes.
    • Search Functionality:
  • Quick Search: Users can search through their contacts quickly and efficiently using the built-in search feature.
    • User Management:
  • Logout and New Registration: Users can log out of their account and register a new account if needed, ensuring a flexible and user-centric experience.
    • User Registration and Login:
      1. New users can sign up by providing their basic information or choose to log in using their Google or GitHub accounts.
      2. Upon successful login, users are directed to their dashboard, where they can start managing their contacts.
    • Search and Navigation:
      • The search bar helps users quickly find specific contacts by typing in names or other relevant details.
      • Users can navigate through their contacts seamlessly, ensuring an efficient management experience.
    • Managing Contacts:
      • Users can add new contacts by filling out a simple form with the contact's details.
      • Contacts are displayed in a list format, allowing users to view and access them easily.
      • The favorite marking feature enables users to prioritize certain contacts, making them readily available.
      • Detailed contact pages allow users to store and view additional information and images related to each contact.
    • Account Management:
      • Users can log out of their current session with a single click.
      • The application allows for new user registrations, providing flexibility for users who need to manage multiple accounts.
